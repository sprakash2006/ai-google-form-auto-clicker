# Mock Google Form Auto Clicker

## Description
A Chrome extension that demonstrates DOM automation on Google Forms.
It auto-fills text fields and selects MCQ options for mock or self-created forms.

## Features
- Auto-fills name and roll number
- Auto-selects MCQ options (demo mode)
- Human-like delay
- Submit button intentionally disabled

## Tech Stack
- JavaScript
- Chrome Extensions (Manifest V3)
- DOM Manipulation

## Ethical Note
This extension is designed ONLY for mock or practice forms.
It does not submit responses and should not be used in live examinations.

## How to Run
1. Open chrome://extensions
2. Enable Developer Mode
3. Click "Load unpacked"
4. Select the project folder
